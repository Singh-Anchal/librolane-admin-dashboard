# Librolane Admin Dashboard

This repository contains the admin dashboard and ERP system for the Librolane project.

## Tech Stack
- Laravel
- PHP
- MySQL
- Vite
- Blade / Admin UI

## Setup Instructions
1. Clone the repository
2. Run `composer install`
3. Create `.env` file and configure database
4. Run migrations
5. Start the development server

## Notes
- `.env`, `vendor`, `node_modules`, and build files are ignored for security.
